Intro

This project is a prime example of using BrainTouch and Python to build an EEG focus and intention recognition system. The aim is to demonstrate how to quickly and effectively structure an EEG-BCI framework using the BrainTouch device. EEG bandpower will serve as the indicator of intention level, as high rhythm power increases when someone focuses their mind on imagining or intending to do something. Additionally, ERD/ERS may be a better way to achieve this goal. We use PyQt5 as the presentation, you can easily replace it with your stimulation.

Dependency

  • Python 3.6+
  • BrainTouch & BrainTouch lab SDK
  • PyQt5

Description

There are two .py files in this project. BrainTouch_socket.py is the standard implementation of the BrainTouch data stream. We use the class BrainTouch_socket.DataStream to get real-time EEG raw data in the BrainTouch_Demo, which includes preprocessing of the EEG raw data and presentation of paradigms. As an interactive sample, we have a Pool game where participants are supposed to focus on imagining hitting the white ball. The program will detect changes in the bandpower of their EEG signal as well as their neural activation. Then, it switches to the successfully hit screen.
